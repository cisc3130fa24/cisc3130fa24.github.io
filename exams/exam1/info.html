Types of questions

The following list covers all types of questions that might be asked. The actual exam that you take will likely not include all of the listed types.
  • Write a basic class, like TimeSpan from HW 2.
  • Write equals and hashCode methods for a class.
  • Given a class, make the class implement Comparable.
  • Given an array or List, write code to sort it by writing a Comparator and passing it to a sort method of the Java library.
  • Writing methods of BoundedList and ArrayBoundedList from HW 4 (partially based on HW 3).
  • Writing generic methods, including with wildcards and the extends and super keywords.
  • Given a generic method and a list of potential arguments to the method, indicate which ones are legal and which aren't.
  • Given a sorted array, trace the execution of binary search, without being provided with the code.
  • Given an array, trace the execution of the following sorting algorithms, without being provide with the code: selection sort, bubble sort (the optimized version), insertion sort, merge sort, and quicksort.
  • Given a piece of code, write the running time as a function of n using big-Oh notation.
  • Know the meaning of Iterable, Collection, SequencedCollection, and List in the Java Collections Framework.
  • On List and ArrayList:
    • Know the following methods and constructors well:
      • The following methods of the List<E> interface:
        • int size()
        • boolean isEmpty()
        • void add(int index, E element)
        • boolean add(E e)
        • default void addFirst(E e)
        • default void addLast(E e)
        • E get(int index)
        • default E getFirst()
        • default E getLast()
        • E set(int index, E element)
        • boolean contains(Object o)
        • int indexOf(Object o)
        • int lastIndexOf(Object o)
        • E remove(int index)
        • boolean remove(Object o)
        • default E removeFirst()
        • default E removeLast()
        • void clear()
        • default void sort(Comparator c)
        • List<E> subList(int fromIndex, int toIndex)
        • default List<E> reversed()
        • static List<E> of(E... elements)
        • static List<E> copyOf(Collection<? extends E> coll)
        • Iterator<E> iterator() and the following methods of the Iterator<E> interface:
          • boolean hasNext()
          • E next()
          • default void remove()
      • The following constructors of ArrayList<E>:
        • public ArrayList()
        • public ArrayList(int initialCapacity)
        • public ArrayList(Collection<? extends E> c)
      • The following List-related methods:
        • public static <T> List<T> asList(T... a)
        • public static <T> List<T> unmodifiableList(List<? extends T> list)
    • Given code involving Lists or ArrayLists or Iterators obtained from them, determine what gets printed.
    • Given code involving a List or ArrayList or an Iterator obtained from one, determine the running time as a function of n (where n is the size of the list) using big-Oh notation.
