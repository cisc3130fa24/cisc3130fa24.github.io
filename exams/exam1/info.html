Types of questions

The following list covers all types of questions that might be asked on the first exam. The actual exam that you take will likely not include all of the possible question types.
  • Write a basic class, like TimeSpan from HW 2.
  • Write equals and hashCode methods for a class.
  • Given a class, make the class implement Comparable.
  • Given an array or List, write code to sort it by writing a Comparator and passing it to a sort method of the Java library.
  • Writing methods of BoundedList and ArrayBoundedList from HW 4 (partially based on HW 3).
  • Writing generic methods, including with wildcards and the extends and super keywords.
  • Given a generic method and a list of potential arguments to the method, indicate which ones are legal and which aren't.
  • Given a sorted array, trace the execution of binary search. (The code will not be provided to you.)
  • Given an array, trace the execution of the following sorting algorithms: selection sort, bubble sort (the optimized version), insertion sort, merge sort, and quicksort. (The code will not be provided to you.)
  • Given a list of functions representing various running times, order them from best (fastest) to worst (slowest). If two functions are equal using big-Oh notation, indicate that.
  • Given a piece of code, write the running time as a function of n using big-Oh notation.
  • Given code involving Lists, ArrayLists, or Iterators obtained from them, determine what gets printed.
  • Given code involving a ArrayList, or an Iterator obtained from one, determine the running time as a function of n (where n is the size of the list) using big-Oh notation.

Items to know from the Java Collections Framework

  • Know the basic meaning of Iterable, Collection, SequencedCollection, List, and ArrayList, and know which inherits from which.
  • Methods of the List<E> interface:
    • int size()
    • boolean isEmpty()
    • void add(int index, E element)
    • boolean add(E e)
    • default void addFirst(E e)
    • default void addLast(E e)
    • E get(int index)
    • default E getFirst()
    • default E getLast()
    • E set(int index, E element)
    • boolean contains(Object o)
    • int indexOf(Object o)
    • int lastIndexOf(Object o)
    • E remove(int index)
    • boolean remove(Object o)
    • default E removeFirst()
    • default E removeLast()
    • void clear()
    • default void sort(Comparator c)
    • List<E> subList(int fromIndex, int toIndex)
    • default List<E> reversed()
    • static List<E> of(E... elements)
    • static List<E> copyOf(Collection<? extends E> coll)
    • Iterator<E> iterator() and the following methods of the Iterator<E> interface:
      • boolean hasNext()
      • E next()
      • default void remove()
  • Constructors of ArrayList<E>:
    • public ArrayList()
    • public ArrayList(int initialCapacity)
    • public ArrayList(Collection<? extends E> c)
  • The following method of the Arrays class:
    • public static <T> List<T> asList(T... a)
  • The following method of the Collections class:
    • public static <T> List<T> unmodifiableList(List<? extends T> list)
